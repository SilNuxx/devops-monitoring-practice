# devops-monitoring-practice
Практический учебный проект, демонстрирующий процесс создания и автоматизации системы мониторинга с использованием Grafana, Loki, Tempo, Prometheus и Kubernetes. Цель проекта — ознакомление с методологией построения эффективных систем мониторинга и получение практических навыков работы с этими инструментами.
